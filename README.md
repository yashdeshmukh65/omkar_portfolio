# My Portfolio

This is a personal portfolio project built using React. It showcases my educational projects and provides information about me.

## Project Structure

The project is organized as follows:

```
my-portfolio
├── src
│   ├── components
│   │   ├── Header.tsx
│   │   ├── Footer.tsx
│   │   ├── Navigation.tsx
│   │   └── ProjectCard.tsx
│   ├── pages
│   │   ├── Home.tsx
│   │   ├── Projects.tsx
│   │   └── ProjectDetail.tsx
│   ├── types
│   │   └── index.ts
│   ├── App.tsx
│   ├── App.css
│   └── index.tsx
├── public
│   └── index.html
├── package.json
├── tsconfig.json
└── README.md
```

## Features

- **Home Page**: Displays current information about me, including my bio and skills.
- **Projects Page**: Lists all my educational projects with a brief description and links to detailed views.
- **Project Detail Page**: Provides detailed information about each project.
- **Responsive Design**: The portfolio is designed to be responsive and accessible on various devices.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-portfolio
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

5. Open your browser and go to `http://localhost:3000` to view the portfolio.
